<h1 align='center'>Portfolio</h1>

<p>This is the repository of my personal <a href='https://marcterradas.vercel.app'>portfolio</a></p>

### Project setup

```
pnpm i
```

### Compiles and hot-reloads for development

```
pnpm run dev
```

### Lint code

```
pnpm run lint
```

### Lint styles

```
pnpm run lint:styles
```

### Run unit test

```
pnpm run test:unit
```
